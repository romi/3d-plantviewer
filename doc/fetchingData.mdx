# Fetching data

This section is dedicated to explain how to make requests to the server
in order to fetch the data necessary to display scans.

## Introduction

Most of the things related to server requests are based in `flow/scans`. There 
are a bunch of hooks there made to fetch data from the server.

The next sections explains and describes what each hook does.

## Hooks

<div>
{
  Object.keys(acc).map((d) => {
    const res = getArgs(acc[d])
    return <>
    <h3> {d} </h3>
    <h4> Parameters </h4>
    {
      (() => { 
      return (res && res.length && res[1]
        ? <table style={{
      border: '1px solid black',
      borderCollapse: 'collapse'
    }}>
      <thead>
        <tr>
          <th> name </th>
          <th> default value </th>
        </tr>
      </thead>
        <tbody>
        {
          (() => {
          return (Array.isArray(res)
            ? res.map((d) => {
            return <tr>
                <th style={{ border: '1px solid black' }}> {d[0]} </th>
                <th> {d.length > 1 ? d[1] : 'None'} </th>
              </tr>
          }) : null)
        })()
      } 
      </tbody>
    </table> : <p> None </p>)
  })()}
  </>
  })
}
