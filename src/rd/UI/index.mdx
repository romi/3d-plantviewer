# UI Components

The goal of this page is to present and reference different reusable UI 
components in the project, and detail their usage.

## MenuBox

### Description

This component is a basic UI component for displaying a floating box, and 
displaying it only when a condition is met.

### Props
<Props of={MenuBox} />

### Exemple

The following exemple shows how you can use a `MenuBox` to display something 
when clicking on a button. Every element that is a child of `MenuBoxContent`
is displaying in the pop up box, the rest is displayed as normal.

<Playground>
  { () => { // MenuBoxExemple
  const [activated, setActivated] = useState(false)
  return (
  <MenuBox 
    activate={activated}
    style={{ textAlign: 'center' }}
    onClose={() => setActivated(false)}
    >
    <button
      onClick={() => { setActivated(!activated) }}
      style={buttonStyle}
    >
      Click me to open a box
    </button>
    <MenuBoxContent>
      <p style={paragraphStyle} > I'm the content of the box ! </p>
    </MenuBoxContent>
  </MenuBox>
  )}}
</Playground>

This exemple uses a React *state* to keep track of the state of the MenuBox 
(opened or closed).
